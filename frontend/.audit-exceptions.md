# npm audit — Exceções Documentadas

Vulnerabilidades conhecidas aceitas com mitigação ativa. Revisadas em 2026-03.

## GHSA-5j98-mcp5-4vw2 — glob CLI command injection
- **Pacote afetado**: `glob` via `eslint-config-next` (devDependency)
- **Exploração**: Requer execução do CLI `glob -c <cmd>` com argumentos controlados pelo atacante
- **Mitigação**: Dev dependency — não executado em produção nem exposto a usuários externos
- **Ação**: Aguardar compatibilidade de `eslint-config-next` com `glob >= 10.4.6`

## GHSA-9g9p-9gw9-jx7f — Next.js Image Optimizer DoS
- **Pacote afetado**: `next` 14.x
- **Exploração**: Requer uso do otimizador de imagens do Next.js com `remotePatterns` inseguros
- **Mitigação**: `next.config.js` tem `images: { unoptimized: true }` — otimizador desabilitado
- **Ação**: Não aplicável

## GHSA-ggv3-7p47-pfv8 — Next.js HTTP request smuggling in rewrites
- **Pacote afetado**: `next` 14.x
- **Exploração**: Requer configuração de `rewrites` no next.config.js
- **Mitigação**: Nenhum `rewrites` configurado em `next.config.js`
- **Ação**: Não aplicável

## GHSA-3x4c-7xq6-9pq8 — Next.js unbounded image cache
- **Pacote afetado**: `next` 14.x
- **Exploração**: Requer uso do Image Optimizer
- **Mitigação**: `images: { unoptimized: true }` — otimizador desabilitado
- **Ação**: Não aplicável

## GHSA-h25m-26qc-wcjf — Next.js HTTP request deserialization (RSC)
- **Pacote afetado**: `next` 14.x
- **Exploração**: Requer uso de React Server Components com dados não-confiáveis sem sanitização
- **Mitigação**: Pages usam Client Components (`'use client'`) com dados sempre validados via API
- **Ação**: Monitorar; fix disponível apenas em Next.js 15+ (breaking change)
